# AI 无限画布工作台

这是一个AI Infinite CanvasPro 架构的本地 AI 画布编辑器原型，采用“原生 Web 前端 + Python 本地服务 + Electron 桌面壳”的结构。

## 当前能力

- 无限画布：节点拖拽、缩放、连线、小地图、多画布页
- AI 节点：图片、文本、视频、音频、360 全景、3D 场景、分镜、注释
- 多服务商设置：自定义api
- 本地项目管理：保存、加载、任务历史、素材和工作流占位
- 本地媒体处理：视频封面与音频分离接口预留
- 桌面增强：Electron 启动本地服务、剪贴板、外链、诊断日志接口
